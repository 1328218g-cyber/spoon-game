쿠키는 어떻게 얻나요?
PC에서 get_session_cookies.js를 한 번 실행해 스푼 로그인을 완료하면 session_cookies.json 파일이 생성돼요. 그 파일 내용을 통째로 복사해서 위 칸에 붙여넣고 업로드하면, 이후로는 서버가 PC 없이 알아서 로그인 상태를 유지합니다. 세션이 만료되면 이 과정을 한 번 더 반복하면 돼요.

💰 펀딩 관리

펀딩을 생성하고 관리합니다. 채팅창에서 설정한 명령어로 [명령어] [번호] [숫자]로 적립합니다.

출력 옵션:
⚙️ 펀딩 명령어 및 문구 커스텀
변수: {month}
변수: {index}, {title}, {current}, {goal}, {percent}, {dday}
📋 명령어 안내

· !펀딩 → 진행중인 펀딩 목록 출력
· !펀딩 2 200 → 2번 펀딩에 200 적립 (DJ 전용)
· !펀딩 2 -50 → 2번 펀딩에서 50 차감 (DJ 전용)

🚩 단비 깃발

깃발 목표를 만들고, 자동 적립 또는 수동 적립을 설정하며, !깃발 명령어나 주기 출력으로 상태를 보여줄 수 있습니다.

사용 예시

· !깃발 → 등록된 전체 깃발 상태 출력
· !깃발 1 → 1번 깃발만 출력
· !깃발 1 50 → 1번 깃발에 50 적립 (DJ 전용)
· 자동 적립 선택 시 스푼을 받으면 선물 수량만큼 자동 누적됩니다.

단비 깃발 설정

⚙️ 실드 명령어 커스텀
채팅창에서 이 명령어를 입력하면 실드 조회/관리가 실행됩니다.
변수: {실드}
변수: {amount}=적립 개수, {실드}=현재 실드 / 호환: {icon}, {action}
변수: {amount}=차감 개수, {실드}=현재 실드 / 호환: {icon}, {action}
🔑 실드 조절 권한 관리 (DJ 전용 설정)
※ 여기에 등록된 고유닉 사용자만 실드 적립/차감이 가능합니다. (DJ는 기본 포함)
🛡️ 실드 명령어 안내

· !실드 : 현재 남은 실드 개수를 출력합니다.
· !실드 +5 : 실드를 5개 추가합니다. (DJ/지정 권한자 전용)
· !실드 -3 : 실드를 3개 차감합니다. (DJ/지정 권한자 전용)
